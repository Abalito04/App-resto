Politicas de privacidad

Ultima actualizacion: 16/06/2026

Datos que se recopilan

La aplicacion puede almacenar datos de cuenta, restaurante, productos, pedidos, metodos de pago, deudas de clientes y configuraciones operativas.

Finalidad

Los datos se utilizan para permitir el funcionamiento del panel, registrar operaciones, generar historial, administrar deudas y facilitar exportaciones.

Acceso a la informacion

La informacion se muestra a usuarios autorizados del restaurante correspondiente. Las claves de acceso y API keys deben mantenerse privadas.

Conservacion

Los datos se conservan mientras la cuenta o restaurante permanezcan activos, salvo eliminacion manual o solicitud especifica de baja.

Servicios externos

La aplicacion puede usar servicios externos para correo, hosting, base de datos o integraciones necesarias para operar el sistema.

Contacto

Para consultas sobre privacidad, correccion o eliminacion de datos, contacta al administrador del sistema.
